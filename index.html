Code Sample: Calculating NPV


from math import pow

def npv(rate, cashflows):
    """ rate as decimal (e.g., 0.1 for 10%) """
    return sum(cf / pow(1 + rate, t) for t, cf in enumerate(cashflows))

if __name__ == "__main__":
    # t=0 is initial investment (negative)
    cashflows = [-1_000_000, 300_000, 350_000, 400_000, 450_000]
    print(npv(0.09, cashflows))

Key Financial Formulas

Net Present Value (NPV): \( \displaystyle \text{NPV} = \sum_{t=0}^{T} \frac{CF_t}{(1+r)^t} \)

Return on Equity (ROE): \( \displaystyle \text{ROE} = \frac{\text{Net Income}}{\text{Avg. Shareholders' Equity}} \)

Black–Scholes (Call): \( \displaystyle C = S_0\Phi(d_1) - Ke^{-rT}\Phi(d_2) \)

with \( d_{1,2} = \frac{\ln(S_0/K) + (r \pm \tfrac{1}{2}\sigma^2)T}{\sigma \sqrt{T}} \)
